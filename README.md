# ROP

Retinopathy of Prematurity (ROP) Images Enhancement Codes.

## Installation

```bash
git clone git@github.com:mhyrzt/ROP.git
python3 -m venv venv
# Activate Your Virtual Environment
pip3 install -r requirements.txt
```
